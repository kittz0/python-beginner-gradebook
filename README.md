# python-beginner-gradebook
Python Project for beginners - Gradebook

**Student Gradebook Manager**
A Python project demonstrating the use of Lists and Two-Dimensional Lists to manage academic records. This script automates the process of organizing subjects and grades, updating scores, and merging historical data.

**Project Purpose**
The goal of this project was to practice basic Python list operations to simulate a real-world scenario: managing a school gradebook. It handles data entry, modifications (such as changing a grade from a number to a "Pass"), and combining multiple semesters of data into one master list.

**Features**
**Dynamic List Creation:** Building lists from scratch and combining them into nested (2D) lists.

**Data Manipulation:** Adding new entries and updating existing records using index targeting.

**Mixed Data Types:** Handling both integers (scores) and strings (pass/fail status) within the same structure.

**List Concatenation:** Merging different datasets into a single "Full Gradebook."

**What I Learned**
Through this project, I gained hands-on experience with the following Python methods and concepts:

.append(): Used to add new subject sub-lists to the main gradebook.

.remove(): Used to target and delete specific values from a sub-list (e.g., removing a numerical score).

2D List Indexing: Learned how to access specific items within a list inside another list (using gradebook[row][column] logic).

List Concatenation (+): Combining two lists to create a larger, comprehensive dataset.

Negative Indexing: Identifying that -1 can be used to reference the most recently added item in a list.
